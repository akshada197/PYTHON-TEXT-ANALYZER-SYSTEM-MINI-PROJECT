# PYTHON-TEXT-ANALYZER-SYSTEM-MINI-PROJECT
A simple Text Analyzer built in Python that performs multiple string operations using loops and conditional statements. Great for beginners to practice logic building and string manipulation.
A menu-driven Python application that analyzes a user-entered sentence or paragraph and performs multiple text-based operations such as counting characters, words, sentences, and classifying different types of characters.

This project is ideal for Python beginners who want to strengthen their understanding of strings, loops, and conditional statements while building interview-ready logic.
